# 列表 lists
## 有序列表 ordered lists
以下為有序列表的例子：

```html
<ol>
  <li>apple</li>
  <li>orange</li>
  <li>pear</li>
</ol>
```

## 無序列表 unordered lists
以下為無序列表的例子：

```html
<ul>
  <li>apple</li>
  <li>orange</li>
  <li>pear</li>
</ul>
```

## 自定義列表 Description Lists
以下為自定義列表的例子：

```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>
```

## 嵌套组合 Nested Lists
```html
<ol>
  <li>Fruit</li>
    <ul>
      <li>apple</li>
      <li>orange</li>
      <li>pear</li>
    </ul>
  <li>Drink</li>
    <ul>
      <li>coffee</li>
      <li>tea</li>
      <li>water</li>
    </ul>
</ol>
```
