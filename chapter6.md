# 嵌入工具 iframe

Google Map、YouTube、facebook 還有很多網頁都設有嵌入工具。
只要直接將**嵌入連結**貼上 html 文件內即可。

```html
<iframe src="......."></iframe>
```
## 堂課五

仿照學校網頁中**聯絡我們**，製作一個`html`文本，文本名稱為`iframe.html`。

``` txt
P:/
└── html
    ├── basic.html
    ├── heading.html
    ├── list.html
    ├── table.html
    ├── link.html
    └── iframe.html
```
