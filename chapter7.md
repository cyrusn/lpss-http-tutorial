# 文字元素 formatting

|    標簽    | 用法 |
|------------|------|
| `<b>`  `</b>`      | 粗體 |
| `<em>` `</em>`    | 強調 |
| `<i>`  `</i>`    | 斜體 |
| `<small>` `</small>`  | 小型 |
| `<strong>` `</strong>` | 重點 |
| `<sub>` `</sub>` | 下標 |
| `<sup>` `</sup>`   | 上標 |
| `<ins>` `</ins>`    | 插入 |
| `<del>`  `</del>`  | 刪除 |
| `<mark>` `</mark>`   | 標記 |

例子：
```
<b>Hello</b><br>

<em>Hello</em><br>

<i>Hello</i><br>

<small>Hello</small><br>
```
