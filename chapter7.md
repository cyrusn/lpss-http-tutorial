# 文字元素 formatting

|    標簽    | 用法 |
|------------|------|
| `<b>`      | 粗體 |
| `<em>`     | 強調 |
| `<i>`      | 斜體 |
| `<small>`  | 小型 |
| `<strong>` | 重點 |
| `<sub>`    | 下標 |
| `<sup>`    | 上標 |
| `<ins>`    | 插入 |
| `<del>`    | 刪除 |
| `<mark>`   | 標記 |

## 堂課六

建立一個 `index.html`文本, 內容必須包括以下兩項。
- 所有`html`課業的超連結
- 關於`html`的感想，200字，而感 想中，必須使用以上文字元素。

``` txt
P:/
└── html
    ├── basic.html
    ├── heading.html
    ├── list.html
    ├── table.html
    ├── link.html
    └── index.html
```
